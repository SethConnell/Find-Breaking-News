API 1.0 Documentation





What Our API Does



By sending a simple request to our server, you will get all of the latest news stories collected from sources like Fox News, The Daily Wire, and Conservative Tribune.

All responses from us will be in JSON format.




Getting Started



Getting set up on our API is easy. Once you create an account, you will have the option to generate a developer key.

Developer keys are just a random string of letters and numbers that lets our server know who you are when you make requests - it's like your personal license plate!

Upon getting this key, you will be able to access all data from our website by visiting NewsFinder.io//api/v1.0/news and passing your api key as "key".




Sifting Through The Data



Sadly, we don't yet offer a way to filter through the sources we send back to you through the API.

If you only want 1 source returned, you will have to figure out how to do that yourself.
